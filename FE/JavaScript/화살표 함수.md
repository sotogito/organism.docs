```javascript
//매개변수 없음
const exp1 = function() {console.log('exp1');};
const arrow1 = () => console.log('arrow1');

//매개변수 하나
const arrow2 = a => {
   console.log('arrow2');
   console.log(a);
}

//매개변수 두개
const arrow3 = (a, b) => {
   console.log('exp');
   console.log(a, b);
   return a+b;
}

//단순 반환 함수 ({}, retrun 생략 가능)
const arrow4 = (x,y) => x*y;
```
- 단일 매개변수의 경우 () 생략 가능
- 단일 실행문일 경우 {} 생략 가능
- 별도의 실행 내용 없이 결과값만 존재할 경우 return 생략 가능
