| 방식                | 설명                                  | 실무 선호도    |
| ----------------- | ----------------------------------- | --------- |
| `@RequestParam`   | 단일 파라미터 받을 때                        | 🔹 자주 씀   |
| `@ModelAttribute` | 여러 파라미터를 DTO로 묶어서 받을 때 (기본값처럼 작동)   | ✅ 매우 자주 씀 |
| `@RequestBody`    | JSON 등 Body 전체를 객체로 받을 때 (REST API) | ✅ 실무 대세   |
| `@PathVariable`   | URL 경로 값 받을 때 (`/notice/{id}`)      | ✅ 자주 씀    |
